# 아크노바 점수판 (1:1)

보드게임 **아크노바**의 2인 대전용 마크 카운트 점수판입니다.
좌=플레이어1 / 우=플레이어2, 마크를 터치하면 +1, 점수는 브라우저에 자동 저장됩니다.

- `index.html` — 점수판 본체 (단일 파일, 오프라인 동작)
- `manifest.json`, `icon-*.png` — 아이폰 홈 화면 추가(PWA)용

## 수정 후 배포
코드를 고친 뒤 아래 3줄이면 끝 — Netlify가 자동으로 새로 배포합니다.

```
git add -A
git commit -m "수정 내용"
git push
```
